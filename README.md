## 🛠️ Built With
- HTML5
- CSS3
- [Bootstrap 5](https://getbootstrap.com/) — layout, cards, buttons, responsive grid
- Bootstrap Icons — SVG icons (Apple, Google Play, features)

## ✨ Features
- Animated gradient hero section with app mockup
- Responsive design — mobile, tablet, and desktop
- Features section with icon cards
- Testimonial section with sponsor logos
- Pricing cards (Chihuahua, Labrador, Mastiff plans)
- Gradient footer with navigation links

## 🚀 Getting Started

Clone the repository and open `index.html` in your browser — no build tools or dependencies needed.

```bash
git clone https://github.com/YOUR-USERNAME/tindog.git
cd tindog
```

Then open `index.html` directly in your browser, or use a live server extension (e.g. VS Code Live Server).
